# JavaScript Calculator App

This project is a simple calculator built using HTML, CSS, and JavaScript.  
The calculator performs basic arithmetic operations like addition, subtraction, multiplication, and division.

## Project Objective

The goal of this project is to build a small interactive JavaScript application using DOM manipulation.

## Features

• Perform addition, subtraction, multiplication, and division  
• Clear button to reset the display  
• Interactive buttons using JavaScript  
• Simple and clean user interface  

## Technologies Used

HTML5  
CSS3  
JavaScript (DOM Manipulation)
