# Weather API App

This project is a simple web application that fetches real-time weather data using a public REST API and displays it dynamically on the webpage.

## Objective

The goal of this project is to integrate a public REST API into a web application and display the fetched data using JavaScript.

## API Used

This project uses the OpenWeatherMap API.

API Website:
https://openweathermap.org/api

The API provides weather information such as temperature, humidity, weather condition, and wind speed for any city.

## Features

• Fetch weather data using Fetch API  
• Display temperature and weather condition  
• Search weather by city name  
• Loading and error handling  
• Responsive user interface

## How it Works

1. The user enters a city name.
2. JavaScript sends a request to the OpenWeather API.
3. The API returns weather data in JSON format.
4. The data is displayed dynamically on the webpage.
